Bike Sales Analysis Dashboard
# Project Overview

This project presents an interactive Power BI dashboard designed to analyze bike sales data and identify the key factors influencing customer purchase decisions. The goal is to uncover customer behavior patterns and generate actionable business insights.

# Dashboard Highlights

Total Buyers & Non-Buyers by Region

Customer Demographics (Age, Gender, Marital Status)

Income & Education Analysis

Commute Distance Impact

Family Type & Car Ownership Trends

Key Influencers Analysis

# Key Insights

North America recorded the highest number of buyers.

Middle-aged and single customers show higher purchase probability.

The medium-income category has the highest number of buyers.

Commute distance and region significantly influence purchase decisions.
